Forms

Checkbox

Because many of the elements that make up our custom checkbox are inline, it is important to remember that there should be no whitespace between them. The code used here has already taken care of that, so feel free to copy/paste to make sure.

<div class="checkbox-control">
	<label class="checkbox-label" for="checkbox1">
		<input id="checkbox1" name="checkbox" type="checkbox"><span class="checkbox"></span><span class="checkbox-spacer"></span><span class="label-text">Choice A</span></label>
</div>
